# DM_closure-prediction
[빅콘테스트 2025 AI·데이터 경진대회] AI 데이터분석 분야  
이정수 · 양원선 

---

## 📌 프로젝트 개요
소상공인의 폐업은 개인의 실패를 넘어  
**지역 상권 붕괴와 금융 리스크로 이어지는 구조적 문제**입니다.

본 프로젝트는 가맹점의 매출·운영·상권·소비 특성을 종합해  
**향후 폐업 가능성이 높은 가맹점을 사전에 예측**하고,  
유형별 특성 분석을 통해 **실질적인 대응 인사이트를 도출**하는 것을 목표로 합니다.

---

## 📖 분석 배경
- 📉 지속적으로 증가하는 소상공인 폐업률  
- 🕰️ 기존 정책은 폐업 이후의 사후 대응 중심  
- 📊 데이터 기반 사전 위험 감지 체계의 필요성

---

## 🎯 분석 목표
1. **폐업 위험 예측 모델 구축**  
2. **폐업 위험 요인 구조 분석**  
3. **가맹점 유형 분류 및 패턴 해석**  
4. **정책·금융 관점의 활용 인사이트 도출**

---

## 📊 사용 데이터
- 가맹점 매출 및 운영 데이터  
- 상권 정보 및 외부 지표  
- 소비자 구성 및 이용 패턴 데이터  

> 시점 정렬이 필요한 고차원 다변량 데이터

---

## 🛠️ 분석 방법론

### 데이터 전처리
- 폐업 여부 기준 재정의 및 시계열 정합성 확보  
- 이상치 처리 및 스케일링  
- 데이터 불균형 해소(SMOTE + Tomek Links)

### 파생 변수 설계
- 가맹점 유지기간  
- 최근 매출 평균 및 변화율  
- 동일 업종 매출 비율 변화  
- 상권 활성화 지수  
- 고객 구성 비율

---

## 🤖 모델링
- Stratified K-Fold 교차검증  
- 다수 모델 비교(Logistic Regression, SVM 등)  
- **SVM 모델 최종 선정**

### 모델 성능
- **F1 Score : 0.8955**  
- Recall : 0.9756  
- Precision : 0.8276  

> 폐업 예측 특성상 위험 가맹점 누락을 최소화하는 방향으로 평가

---

## 📊 가맹점 유형 분석 (K-means)
- **경쟁력 약화형**: 매출 변동성 큼, 폐업 위험 높음  
- **안정 유지형**: 장기 운영, 안정적 매출 구조  
- **지속 성장형**: 매출 증가 추세, 확장 가능성 높음  

> 동일한 비폐업 가맹점이라도 위험 구조는 상이
