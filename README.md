# 🏪 DM_closure-prediction  
**[빅콘테스트 2025 AI·데이터 경진대회] AI 데이터분석 분야**  
이정수 · 양원선  

---

## 📌 프로젝트 개요
소상공인의 폐업은 개인의 실패를 넘어  
**지역 상권 붕괴와 금융 리스크로 이어지는 구조적 문제**입니다.

본 프로젝트는 가맹점의 매출·운영·상권·소비 특성을 종합해  
**향후 폐업 가능성이 높은 가맹점을 사전에 예측**하고,  
유형별 특성 분석을 통해 **실질적인 대응 인사이트를 도출**하는 것을 목표로 합니다.

---

## 🎯 분석 목표
1. 폐업 위험 예측 모델 구축  
2. 폐업 위험 요인 구조 분석  
3. 가맹점 유형 분류 및 패턴 해석  
4. 정책·금융 관점의 활용 인사이트 도출  

---

## 🛠️ 분석 방법론 (1차)

### 데이터 전처리 & 변수 설계
- **이상치 처리 및 변수 스케일링**
- **외부 데이터 활용**으로 결측치 처리 
- **SMOTE + Tomek Links**(오버샘플링, 언더샘플링) 적용으로 데이터 불균형 완화  

#### ✔ 핵심 파생 변수
| 카테고리 | 변수 |
|:--|:--|
| 상권 | 상권_활성화지수 |
| 가맹점 정보 | 유지기간_월 |
| 매출 | 물가 대비 민감도, 동일 업종 매출 비율 변화율, 최근 3개월 매출 평균 |
| 소비자 행동 | 남성 50대 고객 비중 |

> **운영 구조와 변화 추세를 반영한 변수 설계에 집중**

### 모델링
- Stratified K-Fold 교차검증 기반 학습  
- Logistic Regression, SVM 등 다수 모델 비교  
- **최종 모델: SVM**

#### 성능
| Metric | Score |
|:--|:--:|
| **F1 Score** | **0.8955** |
| Recall | 0.9756 |
| Precision | 0.8276 |



### 가맹점 유형 분석 (K-means)
| 유형 | 특징 |
|:--|:--|
| **경쟁력 약화형** | 매출 변동성 큼, 폐업 위험 높음 |
| **안정 유지형** | 장기 운영, 안정적 매출 구조 |
| **지속 성장형** | 매출 증가 추세, 확장 가능성 높음 |

> 동일한 비폐업 가맹점이라도 **위험 구조는 서로 다름**


### 기대 효과
- 🚨 폐업 고위험 가맹점 **사전 식별 가능**  
- 🧭 가맹점 유형별 **맞춤형 금융상품 추천 설계 근거 제공**
