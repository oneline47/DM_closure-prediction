## 📌 프로젝트 개요
소상공인의 폐업은 개인의 실패를 넘어  
**지역 상권 붕괴와 금융 리스크로 이어지는 구조적 문제**입니다.

본 프로젝트는 가맹점의 매출·운영·상권·소비 특성을 종합해  
**향후 폐업 가능성이 높은 가맹점을 사전에 예측**하고,  
유형별 특성 분석을 통해 **실질적인 대응 인사이트를 도출**하는 것을 목표로 합니다.

---

## 🎯 분석 목표

2. 폐업 위험 요인 구조 분석  
3. 정책·금융 관점의 활용 인사이트 도출

<h2>🛠️ 모델 방법론 (개선 이후)</h2>

### 1) 데이터 전처리 & 변수 설계
- 데이터 전처리
  - **이상치 처리 및 변수 스케일링**
  - **외부 데이터 활용**으로 결측치 처리 
  - **분기별로 평균, 표준편차 계산** - 계절성·이벤트성 변동 완화

- 변수 설계
  1. **EDA 결과**를 바탕으로 변수 선택
  2. **Mann–Whitney U test**로 설계한 파생변수의 **p-value < 0.05** 검증
  3. **VIF(분산 팽창 계수)** 기준으로 다중공선성 측정 및 해결
 
  | 카테고리 | 변수 |
  |:--:|:--|
  | 상권 | 상권_활성화지수 |
  | 가맹점 | 가맹점 운영개월수 구간_mean, 동일 업종 내 해지 가맹점 비중_std , 배달여부,<br> 배달매출금액 비율_std, 배달매출금액 비율_mean, 객단가 구간_mean, 객단가 구간_std |
  | 소비자 | 여성_합계_mean, 60대_합계_mean, 20대_합계_mean,<br> 30대_합계_mean, 50대_합계_mean, 60대_합계_std |
  > **가맹점의 운영 상태와 변동성을 함께 담는 변수로 구성**


### 2) 모델링 
- 과정
  - EasyEnsemble, RandomForest, Logistic Regression, SVM 등 다수 모델 비교
  - 2023~2024년 8개 분기별로 모델을 개별 학습
  - SMOTE + Tomek Links(오버샘플링, 언더샘플링) 적용으로 데이터 불균형 완화
    - 데이터 분포에 민감한 선형 모델(SVM, LogisticRegression)에만 적용
  - **최종 모델: EasyEnsemble**
    - 그리드 서치
    
- 성능
  | Metric | Score |
  |:--|:--:|
  | **Recall** | **0.7416** |
  | Precision | 0.0563 |
  | F1-score | 0.0292 |



### 3) 아이디어 제안

- **모델링 결과 해석**
  - **EasyEnsemble**로 폐업 확률을 예측 - 위기 종합 스코어
  - **SHAP**으로 예측 결과를 변수별 기여도로 분해 - 주요 위험 요인(TOP 5) 선정  
  > 위기 스코어, 기여도 기반으로 가맹점별 **위험 요인 진단 및 개선 방향**을 제안

- **폐업 위기 감지 웹페이지 -DM(Closure Danger Monitor)** 서비스 제공  
  - 기능 1. 정량화 - 위기 종합 스코어(0~100)  
  - 기능 2. 요인 진단 - 폐업 위험 요인 비교  
  - 기능 3. 실행 연결 - 요인별 해석 및 개선 방향 제시  
  
- 기대효과
  1. 가맹점 관점  
     -조기 위기 수준 감지  
     -전략적 의사결정 가능  
  
  3. 신한카드 관점  
     -가맹점 조기 부실 위험 탐지 및 리스크 관리 고도화  
     -위험 등급 기반 맞춤형 금융 상품 설계  
     -ESG·상생 금융 모델 강화
